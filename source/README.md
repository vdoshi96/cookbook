# Source PDFs

This directory holds the source PDF(s) of *India Cookbook* by Pushpesh Pant (Phaidon, 2010). The PDF itself is **not committed** — the entire `/source/` directory is gitignored — because the book is copyrighted.

To run the backend pipeline locally, place the PDF at `/source/india-cookbook.pdf`.

This README is the only file in this directory that is checked in.
